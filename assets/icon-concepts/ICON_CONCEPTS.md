# SelectPilot icon concepts

Three launch-direction icon concepts for SelectPilot:

- `selectpilot-concept-a.svg`
  - Selection frame + centered execution core
  - Most minimal and most legible at small sizes

- `selectpilot-concept-b.svg`
  - Selection frame + directional transform arrow
  - Best if you want to emphasize execution and structured flow

- `selectpilot-concept-c.svg`
  - Selection frame + trust/shield mark
  - Best if you want privacy and local boundary to dominate the brand

Recommended direction:

- Start from `concept-b` if the brand should emphasize local execution.
- Start from `concept-c` if the brand should emphasize privacy-first trust.
- Keep `concept-a` as the fallback if Chrome toolbar legibility matters most.
